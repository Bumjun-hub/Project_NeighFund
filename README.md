# Project_NeighFund
> "펀딩과 소모임으로 실현하는 지역 공동체 플랫폼"

---

## ✨ 프로젝트 소개
- 지역 주민이 제안하고, 이웃과 함께 공감과 펀딩으로 지역문제를 해결하는 사회서비스 플랫폼입니다.
- 지역문제해결을 우리 모두의 일로
- 소상공인에게는 새로운 수요의 기회로
- 주민에게는 참여와 이웃간의 관계회복의 시작점으로

---

## 🛠 사용 기술 (Tech Stack)
- Frontend : React, React Router
- Backend : Spring Boot 3.4, Spring Security, JPA, JJWT, WEBSOCKET, STOMP 
- Database : MySQL
